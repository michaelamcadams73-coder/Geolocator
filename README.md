# Geolocator App

The Geolocator app is a multifunctional application that enables users to view geographical locations on a Leaflet map. 

## Features
- Multi-location support
- Add comments and photos to locations
- Easy download functionality for saved locations

### Technologies Used
- Leaflet.js
- HTML5
- CSS3
- JavaScript

## Getting Started
To get started, clone the repository and use the following command to install the necessary dependencies:

```bash
npm install
```

## Usage
Run the application by opening `index.html` in your browser.
